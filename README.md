# Club Application - Backend

## Description

Backend of Club Application app.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Important URL
```bash
# Swagger (API Documentation)
$ host:3000/

# Postgresql (Database)
$ host:5432/

# Adminer (Database Management Tool)
$ host:3645
```
